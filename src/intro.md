## About This Project

This repository aims to provide programming solutions in multiple languages, including:

| Category | Languages |
|----------|-----------|
| Classic | QBasic, Visual Basic, Fortran, Batch, Lua |
| Modern | Java, C#, Go, Haskell, Elixir |
| Scripting | Python, Ruby, Perl, PowerShell |
| Systems | C, C++, Assembly |

Each problem follows a consistent naming convention — `0001_hello_world.ext` — so solutions across languages are easy to find and compare.
