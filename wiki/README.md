# Wiki Overview

This wiki collects the extended documentation for the Programming Solutions Universe repository. Use it alongside the main `README.md`, `questions.md`, and `status.md` files for quick navigation, setup help, and language guidance.

## Structure
- **Home**: Entry point with quick links and repository highlights.
- **Getting-Started**: Step-by-step setup and workflow for running solutions locally.
- **FAQ**: Answers to common contributor and learner questions.
- **Language-Guide-Python**: Python-specific setup tips, patterns, and conventions.

## How to use the wiki
1. Start from **Home** to understand the repository layout and pick a problem.
2. Follow **Getting-Started** to set up your environment and run your first program.
3. Consult **FAQ** for common issues like naming, linting, or status tracking.
4. Use language guides for language-specific instructions; Python is provided now.

## Contributing updates
- Edit the files under `wiki/` and open a pull request; keep headings clear and concise.
- Link to existing pages with relative links (for example, `[[Getting-Started]]`).
- Follow repository conventions: keep filenames aligned with `questions.md`, prefer minimal changes, and avoid editing generated artifacts like `index.html`.
